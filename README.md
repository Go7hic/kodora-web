## What is Kodora?

Kodora brings your AI coding sessions into one place. It reads agent history from folders on your Mac, builds a local search index, and lets you browse projects, sessions, and messages without uploading anything to the cloud.

Built for developers who work across multiple agents and want a fast, native way to revisit past conversations.

## Supported Agents

- Codex
- Claude Code
- Cursor
- OpenCode
- Factory
- Pi Agent

## Features

- **Multi-agent browsing** — projects, sessions, and message details in one app
- **Full-text search** — search sessions and messages, jump to matching records
- **Insights** — usage overview, activity wall, and chart summaries
- **Favorites & archive** — mark important sessions and hide the rest
- **Private notes** — add local notes to sessions, included in search results
- **Export & copy** — copy or export session content as Markdown
- **Sensitive data scan** — scan sessions for potential risks with a security overview
- **Local first** — indexes and session data stay on your machine

## Privacy

Kodora only reads agent history from your Mac. Session content is never uploaded. Indexes, favorites, notes, and archives are stored locally.
